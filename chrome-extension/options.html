Sites to Ignore

Sites to Completely Ignore

These hostnames will be completely ignored by the extension. For example, "google.com", will exclude all mail, documents, and search information. "www.google.com" will only ignore searches on google.com

Sites to only report Hostname

For these hostnames, the extension will only report the hostname, without any additional extension. For example, if you include "mail.google.com", the url "https://mail.google.com/mail/u/0/#inbox/14b430d3e97" will be reported as "https://mail.google.com/"
